UniqueValuesMapping¶

Full name: ase2sprkkr.common.unique_values.UniqueValuesMapping

class ase2sprkkr.common.unique_values.UniqueValuesMapping(mapping, value_to_class_id=None)[source]¶

Bases: object

A class, that can map a collection of (possible non-unique) values to a set of unique identifiers.

The instances of the class can be merged to distinct the values, that are the same according to one criterion, but distinct on the other.

..doctest::
>>> UniqueValuesMapping.from_values([1,4,1]).mapping
np.array([1, 2, 1])
>>> UniqueValuesMapping.from_values([1,4,1]).value_to_class_id
{ 1: 1, 4: 2 }
>>> UniqueValuesMapping.from_values([1,4,1]).merge([1,1,2]).mapping
np.array([1, 2, 5])
