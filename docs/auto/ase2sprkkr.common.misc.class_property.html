class_property¶

Full name: ase2sprkkr.common.misc.class_property

class ase2sprkkr.common.misc.class_property(method=None)[source]¶

Bases: object

Decorator that converts a method with a single cls argument into a property that can be accessed directly from the class.

Example

class Cls:
   @class_property
   def cls_property():
       return some_possibly_cached_value

   x = Cls.cls_property
