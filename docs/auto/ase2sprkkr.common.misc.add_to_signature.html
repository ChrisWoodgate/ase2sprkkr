add_to_signature¶

Full name: ase2sprkkr.common.misc.add_to_signature

ase2sprkkr.common.misc.add_to_signature(func, add={})[source]¶

Add the arguments in the <func> function to the list of arguments of the resulting function (as keyword_only arguments) The modified function has to have its arguments defined as in the following example:

def parent(.....):
    ....

@add_to_signature(parent)
def child(*args, new_param, **kwargs):
    print(f'New param is {new_param})
    parent(*args, **kwargs)
