ase2sprkkr.old.calcio.load_parameters(lines)[source]¶

Load parameters.

Notes

The following assumptions should hold: - The section name is the first word on a line. - The sections are separated by empty lines or comments. - Each entry is on a separate line, with the exception of the first entry

following the section name.

Returns

pars – The parameters as {section_name : section_dict}. When a parameter takes no value, the returned value is ‘@defined’.

Return type

dict
