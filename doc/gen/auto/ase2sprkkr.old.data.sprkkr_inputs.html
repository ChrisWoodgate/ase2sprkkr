Module sprkkr_inputs¶

This function contains input templates for various tasks New template include into dict_inp_data. It returns list of strings which can be proceeded by load_parameters Please be carefull, each keyword in given section has to be on the new line .. code-block:: python

from sprkkr.data import sprkkr_inputs task=’scf’ print(get_sprkkr_input(task))
